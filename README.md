# Echo App

| Input | Output |
| ------ | ------- |
| open IRB | Say something: |
| exit | Goodbye! |
| hello, world | 2018-01-09 | 16:26 | You said: 'hello, world'! ... Say something: |

Methods:
.prompt => Say something:
.response => gets.chomp
.render => date | time | You said: message
.exit => if 'exit', end program
